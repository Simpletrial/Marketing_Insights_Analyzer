# Marketing Insights Analyzer

This project analyzes customer feedback using:
- Rule-based analysis
- Azure OpenAI-based AI analysis

It combines both approaches to generate final insights such as:
- Sentiment
- Key themes
- Complaints
- Improvement suggestions

## Technologies Used
- Python
- Azure OpenAI
- dotenv

## How to Run
1. Install dependencies
2. Create a `.env` file with Azure OpenAI credentials
3. Run `analysis.py` or `main.py`

## Input
- feedbacks.txt contains customer feedback data

## Output
- Structured insights printed in JSON-like format
